验证：页面结构测试通过，核心功能在无样式和无脚本时仍有可读内容。

- 在 `data.js` 增加品牌、商务联系方式、五项价格和披露内容。

- 保留现有 19 个工具和 6 个分类。

验证：配置测试和联盟回退测试通过。

## 4. 实现交互

- 同步侧栏、快捷筛选、Dock 和移动导航的分类状态。
- 实现 `⌘/Ctrl + K` 搜索浮层与 `Escape` 关闭。
- 实现本地收藏，存储不可用时安全降级。
- 实现商务邮件生成和微信号复制反馈。
- 实现商业价格卡和免费/加急提交入口。

验证：逻辑单元测试通过，键盘与触控操作可用。

## 5. 实现视觉与响应式

- 使用 CSS 渐变、玻璃层、主窗口、工具卡和 Dock。
- 桌面三列、平板两列、手机单列。
- 添加克制的进入、悬停和 Dock 动效。
- 支持 `prefers-reduced-motion`。

验证：1440、1024、768、390 和 320 像素截图无溢出，主要文字清晰。

验证：线上页面正常加载；若基础统计无法安全启用，则记录为后续配置，不阻塞发布。
