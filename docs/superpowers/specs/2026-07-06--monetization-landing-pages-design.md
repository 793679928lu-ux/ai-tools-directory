# AETHER 变现落地页设计

日期：2026-07-06

## 目标

把现有“商业合作”和“提交工具”入口升级成两个可直接成交的独立页面，让广告主和工具开发者更容易理解价格、位置、规则和联系方式。

成功标准：

- 新增 `advertise.html`：广告合作 / Media Kit 页面。
- 新增 `submit-tool.html`：提交工具 / 加急审核页面。
- 首页商业合作区和提交工具区链接到独立页面。
- Media Kit 页面包含广告套餐、广告位展示样例、适合投放对象、合作流程、风险说明和可复制招商话术。
- 提交工具页包含免费提交、¥99 加急审核、所需资料、审核规则和联系方式。
- `sitemap.xml` 加入两个新页面。
- README 和测试同步更新。
- 不新增支付、数据库、登录、订单后台、第三方脚本或任何密钥。

## 页面设计

### advertise.html

内容结构：

1. Hero：AETHER 广告合作 / Media Kit。
2. 广告产品：精选工具置顶、内容流原生广告、首页横幅、首发组合套餐。
3. 广告位样例：用页面内 mockup 展示横幅、原生广告、置顶卡片。
4. 适合投放的客户：AI 工具、SaaS、小程序、Chrome 插件、独立开发者产品。
5. 合作流程：联系、确认位置、提交素材、微信收款、上线。
6. 招商话术：提供可复制的微信/邮件文本。
7. 免责声明：不保证点击、成交、排名或收录。

### submit-tool.html

内容结构：

1. Hero：提交你的 AI 工具。
2. 免费提交：排队审核，不保证收录。
3. 加急审核：¥99/次，优先审核，不保证通过。
4. 需要资料：名称、网址、分类、一句话介绍、联系方式。
5. 审核规则：合法、真实、可访问、有清晰用途。
6. 联系入口：邮箱和微信。
