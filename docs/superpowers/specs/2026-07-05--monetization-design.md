## 目标

首期成功标准：

- 访客能看懂广告产品、价格、周期和合作规则。
- 潜在客户能通过邮件或微信完成询价。
- 站长可以人工确认订单并通过微信收款。
- 广告内容与普通推荐有明确区分。
- 联系方式、价格和广告目标地址可集中修改。
- 现有搜索、分类、广告位和静态文件兼容性测试继续通过。

## 方案选择

### 首期：直接销售

直接广告和付费加急最适合当前阶段，因为无需等待第三方平台审批，也不依赖大规模流量。成交由邮件或微信发起，站长人工确认库存、素材、排期和收款。

### 同期预留：联盟佣金

工具数据支持可选的联盟链接和联盟标识，但没有真实联盟链接时继续使用官网地址。只有申请通过并取得专属链接后才允许启用，页面同时显示联盟披露。

优先申请 ElevenLabs 和 Gamma。ElevenLabs 官方当前说明新付费用户首年付款可产生佣金；Gamma 的联盟计划需要申请审核。Notion 当前暂停接受新联盟成员，因此不列为首发任务。

### 暂缓：AdSense

当前站点原创内容与访问量较少，先不加入展示广告网络。后续应先增加原创对比、教程和选型内容，再评估 AdSense。

## 页面设计

### 导航与广告入口

- 主导航保留“广告合作”，指向新的 `#advertise` 商业合作区域。
- 首页横幅广告和内容流原生广告统一指向 `#advertise`。
- 广告入口明确显示“广告”或“赞助”，不伪装成编辑推荐。

### 商业合作区域

在工具列表与提交区域之间增加商业合作模块：

- 标题说明受众是主动寻找 AI 工具的中文用户。
- 展示四档广告产品及价格。
- 首发组合套餐作为主推荐。
- 每个套餐提供“邮件咨询”按钮，自动填入套餐名称、价格、周期、产品名称和网址字段。
- 提供“一键复制微信号”按钮，并用非阻塞提示反馈复制结果。
- 显示“不保证流量、点击或成交；排期和素材审核以人工确认为准”。

### 工具提交区域

拆成两个选择：

- 免费提交：发送普通工具提交邮件，按正常顺序审核。
- ¥99 加急审核：发送含“加急审核”的商务邮件，明确付费不保证收录。

### 披露

页脚增加简短披露：

- 标注“广告/赞助”的内容属于付费展示。
- 部分外链未来可能是联盟链接，用户购买后网站可能获得佣金，但不会增加用户价格。

## 数据与代码边界

继续保持无框架静态站点：

- `data.js`：增加 `business` 配置，包含公开联系方式、套餐、免责声明和可选联盟字段。
- `app.js`：负责渲染套餐、生成预填邮件、复制微信号以及显示反馈；不处理收款和订单存储。
- `index.html`：只增加商业合作和披露所需的语义化容器。
- `styles.css`：增加价格卡、联系方式、披露和移动端布局，不重构现有样式。

联盟字段采用可选值。未配置专属联盟 URL 时，工具卡必须继续跳转官网，不展示联盟标识。

## 数据流

1. 页面从 `data.js` 读取商务配置和套餐。
2. `app.js` 渲染商业合作区域。
3. 用户选择套餐后：
   - 邮件方式打开本地邮件客户端，主题和正文已预填；
   - 微信方式复制公开微信号，用户自行添加并沟通。
4. 站长人工确认广告库存、素材、排期和付款。
5. 成交后只需修改 `data.js` 中的广告内容和目标地址并重新部署。

网站不收集或存储访客姓名、电话、邮件、付款信息。

## 访问统计

启用 Vercel Web Analytics 的基础页面浏览统计，用于了解访问量、来源、国家或地区、设备等汇总信息。首期不设置自定义用户画像，不采集提交内容，也不在页面输出任何密钥。

若当前 Hobby 计划不支持所需的点击事件统计，则只启用基础浏览统计，不为此升级付费计划。

## 错误处理

- 邮件客户端不可用时，页面仍显示可复制的商务邮箱和微信号。
- 剪贴板 API 不可用时，显示微信号并提示用户手动复制。
- 商务配置缺失时，不渲染对应联系方式按钮，避免生成无效链接。
- 联盟链接为空时回退到工具官网。

## 测试

自动化测试覆盖：

- 商务配置包含有效邮箱、微信号和五项价格。
- 邮件主题与正文正确编码。
- 联盟链接为空时回退官网。
- 页面存在商业合作、披露、免费提交和加急审核入口。
- 页面不包含支付密钥、令牌或占位邮箱。
- 现有搜索、分类和静态文件兼容测试继续通过。

人工验证覆盖：

- 桌面和移动端价格卡布局。
- 邮件按钮生成正确内容。
- 微信复制按钮有成功或降级反馈。
- 首页横幅和原生广告能到达商业合作区域。
- GitHub 更新后 Vercel 自动部署，线上页面正常加载。

## 非目标

首期不实现：

- 自动支付、退款、发票或订单后台。
- 用户注册、数据库或广告客户账户。
- 广告库存自动排期。
- 自动批准工具收录。
- 在未获联盟平台批准前伪造或猜测联盟参数。
- 流量、点击、排名或成交保证。
