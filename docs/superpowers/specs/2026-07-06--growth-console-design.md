# AETHER 获客运营后台设计

日期：2026-07-06

## 目标

为 AETHER 增加一个可直接使用的获客运营后台，用于管理每日两条合规图文发帖任务、记录发布动作和实时查看运营数据。

## 边界

- 不自动群发广告。
- 不自动加入客户群。
- 不绕过平台风控或验证码。
- 不使用未授权的平台 API。
- 不收集或公开真实客户隐私数据。
- 第一版数据只保存在浏览器本地，用于实时操作看板；跨设备实时数据库留到下一阶段。

## 页面设计

新增 `growth.html`，设置 `noindex,nofollow`，不加入 sitemap。

页面包含：

1. 今日两条任务：从未发布队列中自动显示前两条。
2. 图文广告卡：每条包含平台、目标客户、图片标题、文案、标签、落地页链接。
3. 图片下载：用浏览器 Canvas 生成 1200x675 PNG。
4. 文案复制：一键复制平台文案和广告链接。
5. 数据录入：曝光、点击、回复、线索数，输入后实时更新。
6. 线索登记：记录客户、来源、意向套餐、状态和备注。
7. KPI 看板：已发布、待发布、曝光、点击、回复、线索、预估金额。

## 测试

- `growth-data.js` 可在无网络环境加载。
- `growth.html` 包含 noindex。
- `growth.html` 不引入远程脚本或样式。
- 运营后台包含每日两条任务、图片下载、文案复制和实时数据看板。
- sitemap 不收录后台页面。
